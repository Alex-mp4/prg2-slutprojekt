Detta är ett kortspel där du väljer vilken klass du ska köra och sedan slåss du mot någon annan! Wow!

Detta spel behöver du öppna i cmd (ifall du gillar att se vad som händer). Öppna Server först, den kommer sedan koppla till din IP. Den som kör client och vill koppla till dig behöver samma IP.

java -jar C:"path"/prg2-slutprojekt-server.jar

java -jar C:"path"/prg2-slutprojekt-client.jar

För att spela spelet behöver du först välja klass för client, sedan server, och sedan client igen. Lite goofy :^)
